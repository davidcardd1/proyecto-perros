Quienes somos

Proyecto Foro es un proyecto realizado en para la materia de Aplicaciones Distribuidas, grupo 4TM2 del periodo 2022/1.

  • Realizado por David Eduardo Cárdenas Ávila y Lisset Cuevas Juárez
  • Usando tecnologías como Django Python y Postgresql
  • Materia dirigida por el profesor Dr. Cesareo Javier Muñoz Rodríguez

Preguntas Frecuentes

¿De qué se trata?

  • Sitio y foro web realizado sobre el framework Django
  • La base de datos es Postgres
  • La realización de posts se logra mediante una herramienta llamada "Widget Teaks"

¿Qué funciones tiene?

  • Sitio:
    • Página inicial
    • Información inicial, presentación del proyecto, contacto, link a foro.
    • Sitio admin (para el control de la web app)
  • Usuarios:
    • Sistema de registro
    • No. Respuestas
    • Log in
    • Recuperación de contraseña
  • Post:
    • Creación
    • Edición
    • Eliminación
    • Respuestas
    • Guardar post
    • Creación con texto enriquecido
  • Foro:
    • Diseño de foro
    • Búsqueda de post y threads (filtrado)
    • Subtemas o subforos
    • Respuestas a posts
